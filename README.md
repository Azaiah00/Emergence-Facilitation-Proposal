# Emergence Global Consulting × GEA Facilitation Proposal

Premium interactive business proposal built with Next.js, Tailwind CSS, and Framer Motion.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Netlify

1. Push this repo to GitHub: [Azaiah00/Emergence-Facilitation-Proposal](https://github.com/Azaiah00/Emergence-Facilitation-Proposal)
2. Log in to [Netlify](https://app.netlify.com/)
3. Click **Add new site** → **Import an existing project**
4. Connect GitHub and select **Emergence-Facilitation-Proposal**
5. Netlify will auto-detect settings from `netlify.toml`:
   - **Build command:** `npm run build`
   - **Plugin:** `@netlify/plugin-nextjs`
6. Click **Deploy site**

Your live URL will be available once the build completes (e.g. `https://your-site-name.netlify.app`).

## Build

```bash
npm run build
npm start
```
